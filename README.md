# Meta Ad Performance & Customer Journey Analytics

### Business Problem

Analyzed **~52,000 advertising campaign interaction records** to evaluate marketing effectiveness and engagement performance.  
The marketing team observed declining engagement and inconsistent conversions but lacked insights into where customers were dropping off in the campaign funnel.

### Analysis

- Performed **data cleaning and validation using SQL and Excel** to ensure consistent campaign identifiers and accurate performance metrics.
- Conducted **Exploratory Data Analysis (EDA)** to evaluate engagement patterns, audience behavior, and time-based performance trends.
- Defined key marketing KPIs including:
  - Click Through Rate (CTR)
  - Cost Per Click (CPC)
  - Conversion Rate
  - Engagement Rate
  - Return on Ad Spend (ROAS)
- Built a **customer journey funnel (Awareness → Engagement → Conversion)** to identify stages where user drop-offs occurred.

### Dashboard

Developed **interactive Power BI dashboards** enabling stakeholders to monitor:

- Campaign performance trends
- Audience engagement patterns
- Funnel conversion rates
- Campaign ROI indicators

### Key Insights

- Identified campaigns with **30–35% higher engagement among specific audience segments**
- Detected major **conversion drop-offs between click and lead submission stages**
- Highlighted campaigns with **high CPC but low conversion efficiency**, indicating targeting optimization opportunities

**Tools:** SQL | Excel | Power BI

### Analysis

- Performed **data validation using Excel and SQL**

### Risk KPIs
